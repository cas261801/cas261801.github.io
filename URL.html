¿Qué es?

  • Un localizador de recursos uniforme (URL) es una dirección web que identifica un recurso específico en Internet. Los navegadores utilizan las URL para obtener recursos como páginas HTML, imágenes o documentos CSS.

    • Partes por las que esta compuesta

    1. Protocolo: Indica el protocolo que se utilizará para transferir datos entre el navegador y el servidor web. Por ejemplo, HTTP o HTTPS.
    2. Nombre de dominio: Indica el servidor web que se solicita.
    3. Ruta al recurso: Indica la ubicación del recurso en el servidor web.
    4. Parámetros: Una lista de pares clave/valor que el servidor web puede utilizar para realizar acciones adicionales.

    SERVIDOR ROOT

    Los servidores llamados root server son los encargados de seleccionar las peticiones, y los servidores DNS establecen la correspondencia entre los nombres de las páginas y las direcciones IP equivalentes.

    PRIMERA PARTE DE LA URL

    La primera parte de un URL, justo atrás de las dos barras, contiene el tipo de protocolo que se utiliza en el intercambio de información, en este caso http: http, documento. ftp, transferencia de archivos. news, noticias usenet. telnet, acceso remoto.

    SEGUNDA PARTE DE LA URL

    La segunda parte del URL, corresponde al nombre del dominio donde se encuentran los datos o el servicio buscado y opcionalmente su puerto de conexión.

    DISEÑO DE LA URL

    Tim Berners-Lee ha dicho que si hubiera sabido que la web iba a ser tan popular como es, debería haber pensado mejor cómo encontrar una alternativa a la parte de una URL. Inicialmente se trataba de código legible por la máquina, pero en la práctica se expone a los usuarios de tal manera que debe ser considerada como un elemento del diseño.
